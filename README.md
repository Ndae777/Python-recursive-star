# Recursive Star Drawing (Python)

## 📌 Description
This project demonstrates recursion in Python by drawing nested stars using the Turtle graphics library. Each star recursively generates smaller stars at its points, creating a fractal-like visual pattern.

## 🚀 Features
- Recursive drawing of nested stars
- Random color generation for each level
- Fractal-style visual output using Turtle graphics

## 🛠️ Technologies Used
- Python
- Turtle Graphics
- Random module

## ▶️ How to Run
1. Ensure Python is installed
2. Run the script:
   python main.py
